# Step by step

## Requirements
python 3.10

## 01 Install FastAPI
pip install fastapi

## 02 Install uvicorn
pip install uvicorn

## 03 Start API
python -m uvicorn main:app --reload 


## Documentation is in http://localhost:8000/redoc

## You can test the API in http://localhost:8000/doc
